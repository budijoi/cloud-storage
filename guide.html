Instalasi

1 SSH ke STB

Buka terminal, ketik:

ssh root@192.168.1.100

Masukkan password root STB.

2 Cek IP STB 
hostname -I

Hasilnya adalah angka IP, misal 192.168.1.100

3 Kirim file dari PC ke STB

Di PC, buka CMD/Terminal:

scp "D:\path\ke\file\*" root@192.168.1.100:/mnt/sdcard/minifileserver/
4 Jalankan installer

Di SSH STB:

bash /mnt/sdcard/minifileserver/install.sh
5 Buka di browser

Buka alamat:

http://192.168.1.100/

atau halaman admin:

http://192.168.1.100/admin.php
